# IG Blendzz — Booking Site

## Quick Deploy to Vercel (FREE)

### Step 1: Get the code on GitHub
1. Go to [github.com/new](https://github.com/new) and create a new repo called `ig-blendzz`
2. Upload all the files from this folder to that repo

### Step 2: Deploy on Vercel
1. Go to [vercel.com](https://vercel.com) and sign up / log in with GitHub
2. Click **"Add New Project"**
3. Import your `ig-blendzz` repo
4. Vercel auto-detects Vite — just click **"Deploy"**
5. Done. You'll get a URL like `ig-blendzz.vercel.app`

### Step 3: Put it in your IG bio
Copy the Vercel URL and paste it in your Instagram bio. That's it.

### Optional: Custom Domain
- Buy a domain like `igblendzz.com` from Namecheap (~$10/year)
- In Vercel dashboard → Settings → Domains → Add your domain
- Follow the DNS instructions Vercel gives you

## Admin Access
